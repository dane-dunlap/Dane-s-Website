Sudoku Project

Portfolio Website

This is a fully functional Sudoku game created using Pygame. The program receives a solvable Sudoku Board via the Sugoku API and uses a backtracking algorithmn to solve the board. Users can click f to visualize how the backtracking algorithmn works!

I created this website from scratch using HTML and CSS. It's not meant for functional purposes but rather just to learn more about web development! The website uses some elements from Boostrap. In the future I would like to make the webpage more responsive and adaptive for mobile views.
